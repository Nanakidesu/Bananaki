# 大阪弁とは？

大阪弁おおさかべんは、関西かんさい地方ちほうで最もっとも代表的だいひょうてきな方言ほうげんのひとつで、漫才まんざいやコメディこめでぃの文化ぶんかと強つよく結むすび付ついています。標準語ひょうじゅんごと比くらべて、音調おんちょうや言葉ことばの言い回いいまわしが大おおきく異ことなります。

# 大阪弁の文法変化ぶんぽうへんか

# 1. 語尾ごびの特徴とくちょう

  • ～ね ⇒ ～な

    • 暑いね ⇒ 暑いな
    • 始まるね ⇒ 始まるな

  • ～よ ⇒ ～で / ～わ

    • 良かったよ ⇒ 良かったで
    • 行くよ ⇒ 行くわ
      「わ」はやや柔やわらかいニュアンス。

  • ～だよね / ～でしょう ⇒ ～やん (か)

    • 嘘だよね ⇒ 嘘やん
    • 見たでしょう ⇒ 見たやん

  • ～だ ⇒ ～や

    • 元気だ ⇒ 元気や
    • 明日だろう ⇒ 明日やろ

  • ～んだ ⇒ ～ねん

    • 海行くんだ ⇒ 海行くねん

  • ～たんだ ⇒ ～てん

    • 行ったんだ ⇒ 行ってん

  • ～ない ⇒ ～へん

    • 行かない ⇒ 行かへん
    • しない ⇒ せえへん

  • ～じゃない ⇒ ～やない / ちゃう

    • 日本人じゃない ⇒ 日本人やない / ちゃう

  • ～ている ⇒ ～てる / ～とる

    • 飲んでいる ⇒ のんでる（男女）/ のんどる（男性）

  • ～てしまった ⇒ ～てもうた

    • 寝てしまった ⇒ 寝てもうた

# 2. 命令形めいれいけいと禁止形きんしけい

  • 命令形：辞書形を長音ちょうおんに

    • 見ろ ⇒ みい
    • 書け ⇒ 書きい
    • しろ ⇒ せえ

  • ～ないで（ください） ⇒ ～んといて（ください）

    • 行かないで ⇒ いかんといて
    • しないでください ⇒ せんといてください

# 3. 助詞じょしや縮約しゅくやく

  • 助詞（は／が／を／へ／に／で）を省略しょうりゃくすることが多い。
  • 原因げんいんを表あらわす「～から」 ⇒ 「～さかい」
    • 暇だから ⇒ 暇やさかい

# 学習がくしゅうのコツ

  • 標準語ひょうじゅんごの例文れいぶんを大阪弁おおさかべんに変換へんかんして練習れんしゅう。

    • 標準語：本当ですか？
    • 大阪弁：ほんまか？

  • 名探偵めいたんていコナンの服部平次はっとりへいじの台詞せりふを聞きいて発音はつおんを真似まねする。

# 結語けつご

大阪弁おおさかべんは単たんなる言葉ことばの違ちがいではなく、大阪おおさかや関西かんさいの文化ぶんかや人情にんじょうを映うつす鏡かがみです。
まずは なんでやねん！ から始はじめて、大阪弁おおさかべんの世界せかいに浸ひたってみましょう。
