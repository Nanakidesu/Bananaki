# APEX 中国語用語（随時更新中）

この度で、「APEX Legends」で結構使われる頻度が高い中国語のゲーム用語を、分かりやすく日中両言語で紹介させていただきます。
中国語の発音を片仮名で注音し、日常会話やゲーム内で役立つフレーズを解説、APEX プレイヤーのコミュニケーションをサポートします。
随時更新できますので、何か補足したいやつがあれば気軽いで連絡してくださいね。ぜひご活用ください！

# 🗣️ 中文の基本挨拶・リアクション集

APEX 中や、チャットで使いやすい中国語の挨拶や感情表現をまとめました。仲間との軽いやりとりや、盛り上がる時にも使ってみましょう！

# 🔸 挨拶・定番フレーズ

  • 你好ニーハオ（nǐ hǎo）＝ こんにちは
    ゲーム開始時やチャットの最初に挨拶するときに使います。

  • 哈喽ハーロウ（hā lóu）＝ ハロー
    フレンドリーに軽く挨拶したいときに使います。

  • 早上好ザオシャンハオ（zǎo shàng hǎo）＝ おはよう
    朝の挨拶として使います。

  • 晚上好ワンシャンハオ（wǎn shàng hǎo）＝ こんばんは
    夜の挨拶として使います。

  • 拜拜バイバイ（bāi bāi）＝ バイバイ
    別れの挨拶として気軽に使います。

  • 再见ザイジエン（zài jiàn）＝ またね
    な別れの挨拶として使います。

# 🔸 お礼・謝罪

  • 谢谢シエシエ（xiè xie）＝ ありがとう
    感謝を伝えたいときに使います。

  • 多谢啦ドゥオシエラ（duō xiè la）＝ マジ助かる！
    特に感謝の気持ちを強調したいときに使います。

  • 不客气ブークーチー（bú kè qi）＝ どういたしまして

  • 辛苦了シンクーラ（xīn kǔ le）＝ お疲れ

  • 对不起ドゥイブチー（duì bu qǐ）＝ ごめん

  • 没事メイシー（méi shì）＝ 大丈夫

# 🔸 代名詞（人称代名詞）

  • 我ウォー（wǒ）＝ 私・僕
    自分を指すときに使います。

  • 你ニー（nǐ）＝ あなた・君
    相手を指すときに使います。

  • 他ター（tā）＝ 彼
    第三者（男性）を指すときに使います。

  • 她ター（tā）＝ 彼女
    第三者（女性）を指すときに使います。

  • 我们ウォーメン（wǒ men）＝ 私たち
    自分を含む複数のグループを指すときに使います。

  • 你们ニーメン（nǐ men）＝ あなたたち
    相手側の複数を指すときに使います。

  • 他们ターメン（tā men）＝ 彼ら
    第三者（男性含む複数）を指すときに使います。

  • 她们ターメン（tā men）＝ 彼女たち
    第三者（女性複数）を指すときに使います。

# 🔸 指示詞（これ・それ・あれなど）

  • 这ヂェ（zhè）＝ これ
    近くのものや話題に出ているものを指すとき。

  • 那ナー（nà）＝ それ・あれ
    遠くのものや話題から少し離れたものを指すとき。

  • 这里ヂェリー（zhè lǐ）＝ ここ
    自分がいる場所や近くの場所を指すとき。

  • 那里ナリー（nà lǐ）＝ そこ・あそこ
    遠くの場所や相手側の場所を指すとき。

# 🔸 程度・大小・数量を表す言葉

  • 很ヘン（hěn）＝ とても
    形容詞の前につけて程度を強調するとき。

  • 太タイ（tài）＝ すごく・〜すぎる
    強い程度を表すとき。

  • 非常フェイチャン（fēi cháng）＝ 非常に
    とても強い程度を表すとき。

  • 一点イーディエン（yì diǎn）＝ 少し
    少量や少しの程度を表すとき。

  • 大ダー（dà）＝ 大きい
    サイズや程度が大きいとき。

  • 小シャオ（xiǎo）＝ 小さい
    サイズや程度が小さいとき。

  • 多ドゥオ（duō）＝ 多い
    数量が多いとき。

  • 少シャオ（shǎo）＝ 少ない
    数量が少ないとき。

  • 全部チュァンブ（quán bù）＝ 全部
    すべてを指すとき。

  • 很多ヘンドゥオ（hěn duō）＝ たくさん
    多量を強調するとき。

これで「你 太 强了！」（君、強すぎ！）とか、「我 有 一点 紧张」（ちょっと 緊張してる）、「他们 都 很 厉害」（彼らみんなすごい）みたいな組み合わせがしやすくなるはずです。
